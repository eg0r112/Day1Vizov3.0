# Краткая инструкция по использованию:
1. Скачайте репозиторий гитхаб
2. Запустите Docker Desktop
3. Запустите контейнер:
в консоли пропишите команду - "docker-compose up -d --build"  
чтобы остановить контейнер используйте команду - "docker-compose down"
4. Запустите приложение - "dotnet run"
5. Откройте swagger по адресу из консоли
6. Зарегестрируйтесь и выберите роль Admin или User
7. Войдите в аккаунт и получите JWT токен
8. Вставьте полученый JWT токен в графу Authorization в правом верхнем углу
# ВАЖНО
перед JWT токеном укажите Bearer
например: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWI.....

В зависимости от того какую роль вы выбрали для вас откроются разные возможности:  
Admin - может взаимодействовать со всеми HTTP методами  
User - только Get запросы  
Бд приложения будет пуста, поэтому вам предстоит вручную заполнить её!  
Так же в приложение предусмотрено сохранение логов в текстовые файлы в папке logs
